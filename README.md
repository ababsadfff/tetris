# tetris
一个基于java的俄罗斯方块，有一下优势
1. 基于位运算的图形表示
使用 16位整数 表示每种俄罗斯方块的形状（4×4 网格）

通过位掩码技术（0x8000）进行方块像素的快速检测与渲染

支持 19种不同的方块形态，涵盖所有经典俄罗斯方块的旋转变化

2. 高效的数据结构设计
26×12 的二维数据数组：维护游戏区域的状态信息

边界检测机制：通过数据层与表现层的分离，实现碰撞检测

行消除算法：采用向下覆盖的方式实现行的消除与分数累加

3. 实时交互系统
实现 KeyListener 接口，支持键盘实时控制

完整的移动逻辑：左移（a）、右移（d）、加速下落（s）、形态变换（w）

边界检测与碰撞算法，确保方块的合法移动
